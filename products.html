TRAQC MODEL I

Inline printed electronics quality control system

Features:

  • Inline
  • At-Speed
  • Realtime
  • Retrofit
  • Contactless
  • Conductivity detection
  • Thickness detection
  • Ink Spreading detection

Description:

TRAQC MODEL I is a solution that allows user to get live information about their printing process without slowing it down or adding any extra out of line verification. The system combines terahertz waves and printed metamaterial to allow for a contactless inspection of key parameters, namely conductivity, thickness and spreading of the printed ink. The system comes with a touchscreen display allowing line operators to view the inline data without any compatibility issues with existing equipment.
